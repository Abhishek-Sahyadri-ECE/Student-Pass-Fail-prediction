# 🎓 Student Pass/Fail Prediction

## 📌 Project Overview
This project predicts whether a student will pass or fail based on:
- Study Hours
- Attendance
- Previous Scores

## ⚙️ Technologies Used
- Python
- Pandas
- Scikit-learn
- Matplotlib
- Seaborn

## 🚀 Features
- Data preprocessing
- Logistic Regression model
- Accuracy evaluation
- Confusion matrix visualization

## 📊 Output
- Predicts Pass/Fail
- Displays accuracy and confusion matrix

## ▶️ How to Run
1. Install libraries:
   pip install pandas matplotlib seaborn scikit-learn

2. Run:
   python Student_Pass_Fail_Predictor.py
